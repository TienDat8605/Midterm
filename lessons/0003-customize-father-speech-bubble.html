Unity story system · Lesson 3

Make the dialogue sound and look like your game

Dialogue content belongs to FatherIntroduction. Typography belongs to DialogueText. Bubble artwork and size belong to BubbleBackground and SpeechBubble.

Change the spoken lines

  1. Open Assets/Scenes/NewMap/Map1.unity.
  2. Select NPCs/FatherNPC.
  3. In Father Introduction → Lines, change the array size and edit each element.

Do not type permanent dialogue into the DialogueText text field. The script replaces that field at runtime.

Change typewriter pacing

  • Initial Delay: wait after the local slime spawns.
  • Seconds Per Character: lower is faster.
  • Pause Between Lines: reading time before replacement.
  • Final Display Time: how long the final message remains.

Change the typography

Select SpeechBubble/DialogueText. Adjust Font Asset, Font Size, Font Style, Vertex Color, alignment, wrapping, and its Rect Transform. Keep at least 20 UI units between text and the decorative border.

Change the bubble

Select SpeechBubble/BubbleBackground to replace its Source Image or adjust its 350 × 140 Rect Transform. Select the parent SpeechBubble to move or scale the complete bubble above Father.

Safe visual preview

  1. Temporarily enable SpeechBubble in the Hierarchy.
  2. Enter sample text in DialogueText and tune the layout.
  3. Before saving, clear the sample text and disable SpeechBubble again.
  4. Run through the normal game flow to test all three real lines.
