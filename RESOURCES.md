# Unity NPC Dialogue Resources

## Knowledge

- [Unity: Camera.ScreenToWorldPoint](https://docs.unity3d.com/6000.3/Documentation/ScriptReference/Camera.ScreenToWorldPoint.html)
  Converts the mouse screen position into the 2D world position used to test NPC clicks.
- [Unity: Physics2D.OverlapPointAll](https://docs.unity3d.com/6000.3/Documentation/ScriptReference/Physics2D.OverlapPointAll.html)
  Documents point-based 2D collider selection and layer filtering.
- [Unity: Canvas](https://docs.unity3d.com/6000.3/Documentation/Manual/UICanvas.html)
  Primary reference for building a screen-space dialogue panel.
- [Unity: Sprite Editor](https://docs.unity3d.com/6000.3/Documentation/Manual/sprite/sprite-editor/sprite-editor-landing.html)
  Primary reference for changing sprite slicing and pivots before scene placement.

## Wisdom (Communities)

- [Unity Discussions](https://discussions.unity.com/)
  Use for engine-version-specific UI, sprite import, and Input System troubleshooting.
